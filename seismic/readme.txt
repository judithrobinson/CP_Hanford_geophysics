This directory contains seismic data collected in FY22 on the Hanford Site.
The data are stored in SEGY format with the *sgy extension.
GPS data are in the traceheaders and conform to standard SEGY conventions (Norris, M. W. and Faichney, A. K. (Eds.), 2002, SEG-Y rev1 Data Exchange format: SEG, Tulsa, OK.
GPS data are in Washington State Plane, meters.

For locations of each line, see the accompanying map (FY22_Seismic_map.png)
